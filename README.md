# PyChess

Um tabuleiro de xadrez virtual que permite apenas lances legais.

## Stack

Python 3.7.9.

## Configuração
1. Clone este repositório:

```git clone git@github.com:aldeoliveira/pychess.git```

2. Instale as dependências:

```pip install -r requirements.txt```

3. Execute a app:

```python main.py```
